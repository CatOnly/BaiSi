# BaiSi
### 一、简介

仿百思不得姐的 APP
**共有 5 大模块**

1. 精华模块
2. 新帖模块
3. 关注模块
4. 用户个人模块
5. 信息发布模块

**共有 5 大功能**

1. 在线视频播放
2. 在线音频播放
3. 在线图片查看
4. 图片存储
5. 图片剪切


### 二、个人写的简易框架

1. 图片加载进度圆形弧度展示框架
2. 网络流媒体简易音乐播放器框架
3. 网络流媒体简易视频播放器框架
